FragebogenPi – lokale Patientenaufnahme auf dem Raspberry Pi.

Anamnesebögen, Patientenfotos und Befundbilder mit GDT-Import und Rückschrieb in die Praxesoftware.
Vollständig offline. Vollständig isoliert. Open Source.

GitHub Dokumentation
▶ Installation
wget https://fragebogenpi.de/install.sh
chmod +x install.sh
sudo bash install.sh
  • Raspberry Pi ab Version 3 (Plastikgehäuse empfohlen – wichtig für WLAN)
  • Raspberry Pi OS (Lite empfohlen)
  • High-Endurance-SD-Karte
  • iPad für Anamnesebögen
  • optional: Smartphone für Befundfotos
  • optional: iPhone für Patienten-Selfies
  • LAN-Anschluss in der Nähe des Wartebereichs
GDT-Import & Rückschrieb · Anamnesebögen · Befundfotos · Patienten-Selfie · offline · isoliert · Raspberry Pi · Open Source
  1. Patientendaten per GDT an FragebogenPi
  2. Patient füllt Formulare am iPad aus
  3. Rückschrieb der Daten und Bilder ins Praxisverwaltungssystem
