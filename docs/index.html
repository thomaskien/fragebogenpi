fragebogenpi - Open Source Anamnesebogen für die Praxis

Anamnesebögen, Patientenfotos und Befundbilder mit GDT-Import und Rückschrieb in die Praxissoftware.
Vollständig offline. Vollständig isoliert. Open Source.

GitHub Dokumentation
▶ Installation kurze URL, Wrapper auf GitHub 
wget https://fragebogenpi.de/install.sh
chmod +x install.sh
sudo bash install.sh
Man benötigt
  • Raspberry Pi ab Version 3 (Plastikgehäuse empfohlen – wichtig für WLAN-Reichweite)
  • Raspberry Pi OS (Lite empfohlen)
  • High-Endurance-SD-Karte
  • iPad Anamnesebogen (z.B. iPad 7 refurbished, griffiger Silikonüberzug z.B. JETech empfohlen)
  • optional: Smartphone für Befundfotos
  • optional: iPhone für Patienten-Selfies
  • LAN-Anschluss in der Nähe des Wartebereichs (für stabile WLAN-Abdeckung)
GDT-Import & Rückschrieb · Anamnesebögen · Befundfotos · Patienten-Selfie · vollständig offline · vollständig isoliert · Raspberry Pi · Open Source

Ablauf

  1. Patientendaten per GDT an FragebogenPi
  2. Patient füllt Formulare am iPad aus (über vom Praxisnetz Abgeschottetem WLAN über den Raspberry Pi)
  3. Rückschrieb der Daten und Bilder per GDT in die Praxissoftware
